✨ Mon Parcours

Mon aventure dans le monde du code a commencé par une simple curiosité pour la manière dont les sites web prenaient vie. Cette curiosité s'est rapidement transformée en une véritable passion. Formé à la fois à l'USTHB et à l'Université Gustave Eiffel, j'ai acquis une double culture, alliant la rigueur de l'algorithmique fondamentale à une approche créative de la conception web.

Aujourd'hui, je me spécialise dans la création d'applications web complètes, du design d'interface (front-end) à la logique serveur (back-end). Mon objectif est de construire des expériences utilisateur fluides, intuitives et performantes, en transformant des idées complexes en solutions numériques élégantes.

📚 Compétences Détaillées

HTML / CSS

Structuration de pages sémantiques et accessibles, et création de designs modernes et responsives avec Flexbox, Grid et les animations CSS.

JavaScript

Développement d'interfaces dynamiques et interactives, manipulation du DOM et communication avec des API (Fetch/AJAX).

PHP

Création de la logique serveur, gestion des sessions utilisateurs, et interaction avec les bases de données pour des sites dynamiques.

SQL / MySQL

Conception de schémas de bases de données relationnelles et écriture de requêtes complexes pour manipuler et extraire des données.

Python & Java

Utilisation de ces langages pour l'algorithmique, la résolution de problèmes et le développement de scripts ou d'applications back-end.

Git & GitHub

Gestion de versions de code pour le travail en équipe et le suivi des modifications, avec un flux de travail basé sur les branches et les pull requests.
